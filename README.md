## Time Series Analysis — Beginner Step-by-Step Guide (EN / FR)

This repository contains a beginner-friendly guide to time series analysis,
designed to explain the full workflow in a clear and progressive way.

- PDF versions for reading
- Word versions for editing and learning

Topics include stationarity, ARIMA/SARIMA modeling, diagnostics, and forecasting.
